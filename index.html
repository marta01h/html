нTML-документ - это обычный текстовый документ, может быть создан (Блокнот), так и в специализированном, с подсветкой кода (Notepad++, visual Studio Code и т.п.), HTML документ имеет расширение .html.

HTML-документ состоит на дерева HTML-элементов и текста, Каждый элемент обозначается в исходном документе начальным (открывающим) и конечным (закрываюцим) тегом (за редким исключением)

Начальный тег показывает, где начинается элемент, конечный путем добавления слэша / перед именем тега: <имя тега>...</имя тега>. Между начальныя и конечным находится содержиное элемента - контент.

Например:

1. <p>контент</p> - Создаёт параграф

2. <b>Контент</b> - Делает полужирное начертание

3 <i>Контент< - Курсивное начертание
